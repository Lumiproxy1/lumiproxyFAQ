# 静态数据中心代理 · 常见问题解答

## 1. 是否支持城市定位？

不支持

## 2. 什么是静态数据中心代理？

静态数据中心代理是一种由数据中心提供的固定IP地址代理服务。与动态代理不同，静态数据中心代理的IP在使用期间不会变化，适用于需要持续使用同一IP地址的应用场景。

## 3. 静态数据中心代理支持什么代理协议？

静态数据中心代理支持http和socks5协议

## 4. 静态数据中心代理优势？

静态数据中心代理相对于其住宅代理的主要优势是DC代理所展示的速度。这使得此类代理成为业务应用程序的最佳选择，在这些应用程序中，代理速度和性能可能是抓取大量数据期间的最终因素。DC代理的速度提升是使用数据中心服务器进行流量而不是使用速度慢得多的单个ISP连接的结果。

## 5. 静态数据中心支持什么验证方式？

提供简单的身份验证方式，您可以使用用户名/密码进行快速认证。

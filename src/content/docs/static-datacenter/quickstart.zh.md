# 静态数据中心代理 · 快速开始

按照以下步骤在 **lumiproxy** 中设置您的 [Static Data Center Proxies](https://www.lumiproxy.com/user/datacenter/pricing/) 。登录、创建代理区域，并验证账户即可开始使用。

注意静态IP暂时不提供测试服务

:::

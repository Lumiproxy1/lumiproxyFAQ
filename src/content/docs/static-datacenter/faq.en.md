# Static Datacenter Proxies · FAQ

## 1. Is city targeting supported?

No, it is not supported.

## 2. What are Static Datacenter Proxies?

A static datacenter proxy is a proxy service that provides a fixed (static) IP address hosted on datacenter servers. Unlike rotating proxies, the IP address of a static datacenter proxy remains unchanged throughout your usage session, making it ideal for applications that require a consistent identity and a continuous connection under the same IP address.

## 3. Which protocols do Static Datacenter Proxies support?

Static datacenter proxies support both **HTTP** and **SOCKS5** protocols.

## 4. What are the advantages of Static Datacenter Proxies?

The primary advantage of static datacenter proxies over residential proxies is the unparalleled speed that datacenter environments offer. This makes them the ultimate choice for business operations and enterprise applications where connection speed and high performance are critical factors—especially when scraping massive volumes of data. This speed boost is a direct result of routing traffic through high-bandwidth datacenter enterprise servers, rather than much slower home ISP connections.

## 5. What authentication methods do Static Datacenter Proxies support?

They provide a straightforward authentication method; you can use **Username & Password** for quick and seamless authentication.

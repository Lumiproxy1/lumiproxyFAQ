# Static Datacenter Proxy · Introduction

**Static Datacenter Proxy** is designed for a variety of business scenarios. This section helps you understand its features, use cases and architecture.

![](/src/assets/img/Static-data/5.png)
## Features

- Stable and reliable IP pool
- Global coverage across major regions
- HTTP and SOCKS5 supported
- Both username/password auth

## Use cases

- E-commerce and social media account management
- Web scraping and price monitoring
- Ad verification and SEO rank tracking

# iOS

本章介绍 **iOS** 的配置与使用说明。

## 准备工作

- 已获取代理账号
- 已安装对应客户端

## 配置步骤
### 设置-Wi-Fi
- 1.打开 iPhone设置。
- 2.点击Wi-Fi 。
- 3.选择您所连接的网络右侧的信息图标。
- 4.向下滚动到HTTP 代理部分。
- 5.单击“手动” 。
- 6.编辑您的代理设置，例如 服务器 – us.lumiproxy.com, 端口 – 5888。如果网络 IP 未在仪表板中列入白名单，请启用身份验证，并添加您的 用户名 和 密码.
- 7.点击返回Wi-Fi选择窗口。这将保存你的代理设置。
- 8.如果您使用身份验证，可能会出现第二次提示，要求您输入代理用户：密码，如果是，请再次输入。




### Shadowrocke

在将代理集成到Shadowrocket之前，请确保您已经正确设置代理，以保障代理服务能够正常运行。
1. 打开Shadowrocket，点击右上角的“ +”号添加配置
:::tip 💡

若在中国大陆使用，需先使用软路由跳转到非中国大陆的网络环境
:::
![](/src/assets/img/mobile/ios/小火箭1.png)


2. 选择代理类型**只支持HTTP和SOCKS5**，若使用API提取模式，则输入从Lumiproxy复制的主机和端口，若选择的账密认证方式，则还需要填写用户名和代理密码。填写完成后点击**代理通过**，选择设置的**软路由节点、vpn、vps**，最后点击保存
![](/src/assets/img/mobile/ios/小火箭2.png)


![](/src/assets/img/mobile/ios/小火箭3.png)
3. 最后选择配置的文件，打开代理连接，这样就可以开始使用


:::tip 💡

浏览器访问api.ip.cc

:::

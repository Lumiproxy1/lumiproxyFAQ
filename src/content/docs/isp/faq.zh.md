# 长效 ISP · 常见问题解答

## 1. 什么是长效isp?

长效ISP代理是数据中心和住宅 IP 的结合，单个IP使用时间最长可达12小时，自动轮换。具备专属 IP 资源、无限连接数和线程数的特点。

## 2.长效isp与住宅代理区别？

来源：ISP代理源于互联网服务提供商的网络基础设施，而住宅代理则来自分配给个别住户的真实住宅连接。
真实性：ISP代理由服务提供商直接管理，提供可靠性和速度。住宅代理则提供更高级别的真实性，模仿真实的住宅IP地址。
用途：ISP代理适合常规浏览和访问全球内容，而住宅代理更适合需要真实用户行为的任务，如网络抓取和市场研究。
稳定性：ISP代理通常更稳定，而住宅代理的稳定性可能因其来源而有所不同。

## 3. 长效isp代理支持什么代理协议？

长效isp代理支持http和socks5协议类型

## 4. 长效isp验证模式？

支持账密验证

## 5. 长效isp代理是如何工作的？

与其他代理服务器类似，长效ISP代理使用不同的IP地址转发您的请求，同时隐藏您自己的IP地址。 长效ISP代理使用消费者互联网服务提供商的自治系统号码（ASN），但不需要处理终端用户的较差互联网连接

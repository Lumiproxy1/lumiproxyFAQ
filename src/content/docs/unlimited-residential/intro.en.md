# Unlimited Residential Proxies · Introduction

**Unlimited Residential Proxies** are proxy products tailored for various business scenarios. This chapter will help you understand the product features, applicable scenarios, and basic architecture.
![](/src/assets/img/Unlimited/2.png)

## Product Features

* Unlimited IP usage and traffic
* Coverage across 50 regions globally
* Support for both HTTP and SOCKS5 protocols
* Supports both Username & Password and Whitelist authentication methods

## Applicable Scenarios

* Audio/video data harvesting and multimedia streaming scraping
* Large-scale, high-concurrency web scraping and data mining
* Multi-threaded automation tools, botting, and sneaker copping
* Social media marketing automation
* Ad verification and fraud detection

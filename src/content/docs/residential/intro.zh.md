# 住宅代理简介

> Lumiproxy 住宅代理网络：每月 9000万+ 住宅 IP，覆盖 195+ 国家，可作为真实用户进行浏览和数据收集。

Lumiproxy的[住宅代理网络](https://www.lumiproxy.com/pricing/residential/)是我们的核心代理解决方案之一，提供对**每月 9000万+ 住宅 IP**的访问，这些 IP 分布在**全球 195+ 国家**。

与基于数据中心的代理不同，住宅代理通过**真实用户拥有的真实设备**路由流量，这些用户已明确选择加入并因参与网络而获得补偿。这种选择加入模式使您能够使用**真实本地IP**访问网站，使您的流量看起来似乎来自目标位置的真实住宅用户。

由于请求通过真实住宅连接传递，此网络特别适合访问**复杂或高度受保护的网站**，这些网站主动阻止自动化或非住宅流量。因此，您可以浏览网站、查看本地化内容或以与真实人员体验网站方式非常相似的方式收集数据。

## 住宅代理如何工作

当您通过住宅代理网络发送请求时，Lumiproxy将该请求路由通过位于您选择的目标区域中的**真实住宅设备**。

在后台，该平台自动：

* 从目标位置选择适当的住宅IP
* 管理IP可用性和轮换
* 在需要时维持会话稳定性
* 处理网络级可靠性和扩展

由于请求源自真实的家庭或移动连接，网站将流量视为来自真实最终用户，而不是自动化系统或基于服务器的代理。

## 住宅代理的主要优势

住宅代理相比其他代理类型提供几个优势：

* **在具有严格防机器人或反爬虫防护的网站上成功率更高**
* **访问本地化内容**，如地区定价、语言变体和地理受限页面
* **降低被阻止的风险**（与基于数据中心的流量相比）
* **可扩展的基础设施**，支持小型和企业级工作负载

这些优势使住宅代理成为生产级数据收集和网络访问用例的强大选择。

## 地理覆盖和定位

住宅代理网络支持**细粒度地理定位**，允许您根据以下内容路由流量：

* 国家
* 州或地区（如支持）
* 城市

这种级别的定位可实现高度准确的本地化，用于测试、数据收集和内容验证用例。

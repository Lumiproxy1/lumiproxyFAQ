# 住宅代理 · 常见问题解答

## 1. 连接超时怎么办？

请检查本机网络、防火墙以及账号是否欠费；可先用 cURL 排查。

```
curl -x http://username:password@Proxy server:port http://api.ip.cc

curl -x socks5h://username:password@Proxy server:port http://api.ip.cc

```

## 2. 响应报错？
- `200`-响应成功
- `403`-域名封禁
- `470`-没有输入usernane和password
- `490`-该地区没有IP
- `512`-白名单认证错误
- `610`-IP是中国大陆IP(不支持中国大陆网络环境直连)
- `612`-账密错误、子账号流量上限、主账号无剩余可用流量(流量用完或套餐过期)
- `613`-并发数超过限制


## 3. 如何保持同一个IP使用120分钟？

在用户名后追加 `-session-xxxx`，session-(参数)。

:::tip 💡

参数由数字字母（大小写）组成。参数越复杂越稳定

:::

# 住宅代理 · 快速开始

按照以下步骤在 **lumiproxy** 中设置您的 [Residential Proxy](https://www.lumiproxy.com/zh-hans/pricing/residential/) 。登录、创建代理区域，并验证账户即可开始使用。


:::tip 💡 
新用户可以联系客服人员领取512MB,测试服务

:::

[注册](https://www.lumiproxy.com/zh-hans/signup/)

在控制台选择适合您业务规模的套餐并完成支付。

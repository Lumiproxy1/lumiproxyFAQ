# ZiNiao Browser

This chapter covers the configuration and usage instructions for **ZiNiao Browser**.


## ZiNiao Browser Proxy Integration

Follow these steps to set up your LumiProxy proxy within ZiNiao Browser:

### Step 1. Install ZiNiao Browser

Download and install **[ZiNiao Browser](https://www.ziniao.com/)**. Launch the application and log into your account.

### Step 2. Create a Proxy Window

In the left menu, go to **Management System** -> **Device Management** -> **Add Own Device**.

![](/src/assets/img/fingerprint/ziniao/ziniao1-en.jpg)

### Step 3. Configure LumiProxy Proxy Settings

* **Proxy Type**: Choose `HTTP` or `SOCKS5` based on your proxy protocol.
* **IP Address**: Enter `us.lumiproxy.io`.
* **Port**: Use the port number provided in your [LumiProxy Dashboard](https://www.lumiproxy.com/user/dashboard/).
* **Username**: Enter your LumiProxy proxy `username`.
* **Password**: Enter your LumiProxy proxy `password`.

![](/src/assets/img/fingerprint/ziniao/ziniao2-en.jpg)


### Step 4. Test Proxy Connection Connectivity

After filling out all the required information, click **Check Network** to test the connectivity of your proxy IP.

![](/src/assets/img/fingerprint/ziniao/ziniao3-en.jpg)

### Step 5. Launch the Browser

Navigate to the **Account Management** -> **All Accounts** screen. Find the account environment you just configured and click the **Launch** button to begin.

![](/src/assets/img/fingerprint/ziniao/ziniao4-en.jpg)

:::tip 💡
You can visit `api.ip.cc` inside the launched browser session to verify your active IP location details.
:::
